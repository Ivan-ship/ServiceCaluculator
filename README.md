# Telegram Mini App – Service Calculator

Интерактивный калькулятор услуг, реализованный как Telegram Mini App с интеграцией бота.

## 🚀 Функционал

- Расчет стоимости услуг в реальном времени
- Удобный UI (React)
- Интеграция с Telegram Web Apps API
- Отправка заявки пользователем через Telegram-бота
- Гибкая настройка параметров и цен

## 🛠️ Стек

Frontend:
- React (Vite)
- Telegram Web Apps API

Backend:
- Python

## 📲 Как это работает

1. Пользователь открывает Mini App в Telegram
2. Выбирает параметры услуги
3. Получает итоговую стоимость
4. Отправляет заявку
5. Бот получает данные пользователя
